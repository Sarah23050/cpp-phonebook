# C++ User Skills Management System
A console-based phonebook application written in C++ that allows users to register, manage their skills and calculate a salary based on skill levels. The application supports file saving and loading using basic file I/O.


## Features

- Register new users with username, email and location
- Add skills with levels (Beginner, Intermediate, Advanced)
- Edit existing skill levels
- Delete skills
- Display user information and skills
- Calculate salary based on skill levels
- Save and load from file


## Menu Options

1. Register a new user
2. Add new skills
3. Show user information
4. Delete a skill
5. Edit skill level
6. Exit the application

## Technologies Used

- C++
- STL (vector, map)
- File I/O
- Git & Github

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Sarah23050/cpp-phonebook.git
   ```

2. Compile the program:
   ```bash
   g++ phonebook.cpp -o phonebook
   ```

3. Run the program:
   ```bash
   ./phonebook
   ```
